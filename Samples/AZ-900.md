## Which cloud service model gives the most control over the operating system?
<!-- type: single -->
<!-- tags: cloud-concepts -->

- [ ] SaaS
- [ ] PaaS
- [x] IaaS
- [ ] FaaS

> **Explanation:** IaaS exposes the VM and OS to the customer, giving maximum control while Azure manages the physical hardware.
> **Reference:** https://learn.microsoft.com/azure/

## Which are characteristics of cloud elasticity? (Choose two.)
<!-- type: multiple -->
<!-- tags: cloud-concepts -->

- [x] Resources scale out automatically under load
- [x] You pay only for what you consume
- [ ] Capacity is fixed at provisioning time
- [ ] Scaling requires a support ticket

> **Explanation:** Elasticity means automatic scale-out/in with consumption-based billing.

## Azure Availability Zones protect against a full region outage.
<!-- type: truefalse -->
<!-- tags: resiliency -->

- [ ] True
- [x] False

> **Explanation:** Availability Zones protect against datacenter-level failures within a region, not a full region outage — use multi-region designs for that.

## A `resource group` is a logical container for related Azure resources.
<!-- tags: governance -->

- [x] True
- [ ] False
